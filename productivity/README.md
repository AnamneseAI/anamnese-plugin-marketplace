# Anamnese Productivity

Productivity system for knowledge workers. Daily planning, career growth, knowledge management, and project tracking -- backed by persistent cloud memory.

## Getting Started

1. **Install the plugin** -- `claude plugins add anamnese-productivity`
2. **Authenticate** -- Run `/start` or `/mcp` to sign in via browser
3. **Chat naturally** -- Proactive capture automatically saves important details as you chat

## Commands

| Command | Description |
|---------|-------------|
| `/start` | Onboard, authenticate, and bootstrap your work profile |

## Skills (auto-activated)

| Skill | What it does |
|-------|-------------|
| Anamnese | Core memory, tasks, goals, notes, and proactive capture |
| Daily Planning | Work day scheduling, time-blocking, calendar integration |
| Career Growth | Professional development, skill building, performance tracking |
| Knowledge Management | Capturing learnings, documenting processes, personal knowledge base |

## Companion MCPs

- **Google Calendar** -- Calendar events for daily planning
- **Microsoft 365** -- Outlook calendar, mail, and more
- **Slack** -- Communication context
- **Notion** -- Docs and wiki access
- **Linear** -- Issue and project tracking
- **Atlassian** -- Jira issues and Confluence pages
- **Asana** -- Project and task tracking
- **Monday** -- Work management and project tracking
- **ClickUp** -- Project management and tasks

You only authenticate the MCPs you want. Unused ones are simply ignored.
