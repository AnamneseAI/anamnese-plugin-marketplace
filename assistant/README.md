# Anamnese Assistant

Life and productivity skills powered by [Anamnese](https://anamneseai.app) persistent cloud memory.

## What It Does

Anamnese Assistant adds domain-specific skills to your AI that activate automatically based on conversation context:

- **Goal Planning** -- SMART framework, milestones, progress tracking across all life areas
- **Financial Awareness** -- Budget frameworks, expense categorization, savings targets
- **Wellness** -- Habit building, exercise, sleep hygiene, mental health check-ins
- **Daily Planning** -- Eisenhower prioritization, time-blocking, calendar integration
- **Career Growth** -- Brag documents, skills assessment, performance review prep
- **Knowledge Management** -- Zettelkasten-inspired personal knowledge base

## Requirements

Requires the **anamnese-core** core plugin to be installed for memory, tasks, goals, and notes.

## Integrations

Pre-configured MCP connections for:
- Google Calendar, Gmail, Microsoft 365
- Slack, Notion, Linear, Atlassian (Jira), Asana, Monday, ClickUp

Connect the ones you use via `/mcp` -- unused integrations are ignored.

## Getting Started

Run `/anamnese-assistant:start` to authenticate and set up your profile.
