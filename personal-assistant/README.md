# Anamnese Personal Assistant

AI-powered personal life assistant. Goal planning, financial awareness, wellness tracking, and life management -- backed by persistent cloud memory.

## Getting Started

1. **Install the plugin** -- `claude plugins add anamnese-personal-assistant`
2. **Authenticate** -- Run `/start` or `/mcp` to sign in via browser
3. **Chat naturally** -- Proactive capture automatically saves important details as you chat

## Commands

| Command | Description |
|---------|-------------|
| `/start` | Onboard, authenticate, and bootstrap your profile |

## Skills (auto-activated)

| Skill | What it does |
|-------|-------------|
| Anamnese | Core memory, tasks, goals, notes, and proactive capture |
| Goal Planning | Track personal goals, life milestones, and aspirations |
| Financial Awareness | Budget decisions, financial goals, spending patterns |
| Wellness | Health habits, exercise, sleep, nutrition tracking |

## Integrations

- **Google Calendar** -- Calendar events for daily planning
- **Gmail** -- Email access

You only authenticate the integrations you want. Unused ones are simply ignored.
